# TeleVault: быстрый запуск на Windows

TeleVault - локальный просмотрщик Telegram exports. Приложение открывает папки Telegram Desktop export, показывает переписки и медиа в браузере и не отправляет данные наружу.

## Как запустить

1. Откройте папку проекта TeleVault.
2. Дважды нажмите `run_windows.bat`.
3. Дождитесь автоматического открытия браузера или откройте URL, который напечатан в консоли.

## Как добавить Telegram export

1. Нажмите `выбрать папку экспорта`.
2. Выберите папку экспорта Telegram или общую папку, внутри которой лежат экспорты с `result.json`.

Приложение работает локально. Ручной ввод пути и старая кнопка `загрузить` не используются.

## Версия

Версия видна в правом верхнем углу интерфейса и в ответе `/api/status`.

Текущая версия: `2.6.7`.
