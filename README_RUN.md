# TeleVault: быстрый запуск на Windows

TeleVault - локальное личное хранилище для экспортов Telegram. Приложение открывает папки Telegram Desktop export, показывает переписки и медиа в браузере и не отправляет данные наружу.

## Как запустить

1. Откройте папку проекта TeleVault.
2. Дважды нажмите `run_windows.bat`.
3. Откройте адрес, который покажет приложение. Обычно это `http://127.0.0.1:8766`.

## Как добавить Telegram export

1. Нажмите `добавить в хранилище`.
2. Выберите папку экспорта Telegram или общую папку, внутри которой лежат экспорты с `result.json`.

## Версия

Версия видна в правом верхнем углу интерфейса и в ответе `/api/status`.

Текущая стабильная версия: `2.5.25`.

Ручной ввод пути и старая кнопка `загрузить` не используются.
