# TeleVault

TeleVault — локальный оффлайн-архив Telegram-переписок для Windows.

Выберите папку экспорта Telegram Desktop, откройте переписки и спокойно читайте сообщения, фото, видео, голосовые, аудио и файлы без интернета.

TeleVault не подключается к Telegram, не требует входа в аккаунт и не синхронизирует данные с облаком. Все остается на вашем компьютере.

---

## Зачем это нужно

Telegram Desktop умеет экспортировать переписки, но читать такие экспорты напрямую неудобно: сообщения, медиа и файлы лежат в папках, а не в спокойном интерфейсе для чтения.

TeleVault превращает выбранный экспорт в удобный локальный архив:

- выбираете папку Telegram Desktop export;
- открываете список переписок из этого экспорта;
- читаете сообщения в удобном виде;
- просматриваете фото, видео, голосовые, аудио, файлы и стикеры;
- ищете нужное внутри выбранной переписки;
- возвращаетесь к последнему открытому архиву при следующем запуске, если папка доступна.

Основная формула простая:

> выбрать папку экспорта → открыть переписку → читать сообщения и медиа оффлайн

---

## Что поддерживается

- full Telegram Desktop JSON exports с несколькими чатами через `chats.list`;
- `left_chats.list`, если Telegram Desktop добавил этот раздел в export;
- single-chat exports с обычным `messages`;
- сообщения, фото, видео, voice, audio, файлы и стикеры;
- replies и отметка edited;
- частые системные события Telegram в понятном виде;
- безопасное отображение базового форматирования текста;
- audio metadata, если Telegram Desktop сохранил performer/title;
- редкий Telegram-контент, например опросы, контакты, геолокации, платежи и игры, отображается как понятные fallback-сообщения;
- аккуратные ошибки для неправильной папки, слишком широкой папки или поврежденного `result.json`.

Сообщения конкретного чата загружаются при открытии чата. Большие архивы из-за этого открываются легче: TeleVault не пытается сразу отрисовать все сообщения всех переписок.

---

## Что умеет

- открывать локальные Telegram Desktop exports;
- показывать список переписок из выбранного экспорта;
- открывать фото и изображения;
- воспроизводить видео;
- воспроизводить голосовые и аудио;
- ставить предыдущее медиа на паузу при запуске другого;
- показывать файлы и документы;
- показывать пересланные сообщения, replies, edited и частые service notices;
- искать внутри выбранной переписки;
- работать оффлайн после выбора папки экспорта.

---

## Приватность

TeleVault сделан как local-first приложение.

- Все хранится на вашем компьютере.
- TeleVault не отправляет переписки во внешние сервисы.
- Вы сами выбираете папку Telegram Desktop export.
- Telegram login не нужен.
- Cloud/sync нет.
- Аккаунтов пользователей нет.
- AI-обработки сообщений нет.
- TeleVault не является Telegram-клиентом.

Приложение работает с уже готовыми Telegram Desktop exports, которые находятся на вашем компьютере.

## Как начать

1. Экспортируйте нужные данные через Telegram Desktop в JSON-формате.
2. Скачайте portable zip из раздела Releases.
3. Распакуйте архив в удобную папку.
4. Запустите `TeleVault.exe`.
5. Нажмите `добавить экспорт`.
6. Выберите папку экспорта Telegram Desktop.
7. Откройте переписку и читайте архив.

Установка не нужна: portable-версия запускается прямо из распакованной папки.

## Платформы

Основной сценарий:

- Windows 10;
- Windows 11;
- portable zip;
- запуск через `TeleVault.exe`.

`run_windows.bat` остается fallback/debug-запуском для разработки и ручной диагностики.

---

## Ограничения

TeleVault — это не Telegram-клиент.

Приложение:

- не выполняет вход в Telegram;
- не синхронизируется с Telegram;
- не получает новые сообщения;
- не отправляет сообщения;
- не работает с bot token;
- не является облачным хранилищем;
- не заменяет Telegram Desktop.

TeleVault читает локальные Telegram Desktop exports, которые уже находятся на вашем компьютере.

## English

TeleVault is a local offline reader for Telegram Desktop exports on Windows.

Choose a Telegram Desktop export folder, open a chat, and read messages, photos, videos, voice messages, audio, files and stickers without internet access.

TeleVault is not a Telegram client. It does not require Telegram login, does not sync with Telegram, and does not upload your data to the cloud. Your archive stays on your computer.
