# TeleVault

**TeleVault — офлайн-ридер экспортов Telegram Desktop для Windows 10/11.**

Он превращает экспорт Telegram в удобное локальное приложение: можно читать переписки, быстро открывать фото, листать медиа, смотреть видео, слушать голосовые и не копаться вручную в папках экспорта.

Это не Telegram-клиент, не облако и не синхронизация. TeleVault работает с уже созданными экспортами Telegram Desktop и хранит всё на твоём компьютере.

## зачем нужен

Обычный экспорт Telegram в HTML/JSON неудобен для долгого просмотра: медиа лежат отдельно, фото и видео приходится открывать по одному, аудио легко превращается в кашу, а общий вид больше похож на техническую выгрузку, чем на нормальную переписку.

TeleVault делает экспорт ближе к обычному мессенджеру:

* фото открываются в удобном просмотрщике и листаются внутри переписки;
* можно перейти только к фото, видео, аудио или файлам, не прокручивая весь чат вручную;
* видео и видеосообщения отображаются прямо в интерфейсе;
* круглые видеосообщения выглядят как привычные “кружочки”;
* голосовые и аудиофайлы удобно слушать внутри чата;
* при запуске нового аудио предыдущее не продолжает играть поверх него;
* файлы из экспорта видны в нормальном списке, а не теряются среди папок;
* переписка читается в спокойном минималистичном интерфейсе.

## privacy / local-first

TeleVault работает локально.

Приложение не требует:

* входа в Telegram;
* bot token;
* облачного аккаунта;
* синхронизации;
* загрузки экспортов на внешний сервер.

Экспорт остаётся там, где ты его хранишь.

## как запустить

1. скачай последнюю версию в разделе **Releases**.
2. распакуй архив `TeleVault-vX.X.X-windows-x64.zip` в отдельную папку.
3. запусти `TeleVault.exe`.
4. выбери папку экспорта Telegram Desktop.
5. открой нужную переписку и медиа.

## какой экспорт нужен

TeleVault рассчитан на экспорт, созданный в Telegram Desktop.

Лучше использовать экспорт в формате **JSON** и сохранять медиафайлы вместе с перепиской.

## платформы

Основной package:

* Windows 10 / 11;
* portable zip, без установки.

Дополнительно может быть доступна legacy-сборка для Windows 7 SP1 x64. Это режим **best effort**, а не основная поддерживаемая платформа.

## ограничения

TeleVault не получает данные из Telegram и не восстанавливает удалённые чаты.

Он открывает только те файлы, которые уже есть в папке экспорта на компьютере.

В проект не входят:

* Telegram login;
* cloud/sync;
* аккаунты пользователей;
* AI;
* теги;
* заметки;
* избранное;
* статистика.
