# TeleVault

TeleVault - локальное оффлайн-хранилище важных Telegram-переписок.

Приложение открывает папки экспорта Telegram Desktop, показывает переписки, медиа и файлы в удобном локальном интерфейсе и не отправляет данные наружу. Архив остается на вашем компьютере.

Актуальная версия: `2.9.7`.

Скачать: `TeleVault-v2.9.7.zip` в [GitHub Releases](https://github.com/serzhberdnyk/televault/releases).

## Быстрый запуск

Основной пакет рассчитан на Windows 10/11 x64.

1. Скачайте `TeleVault-v2.9.7.zip`.
2. Распакуйте zip в отдельную папку.
3. Запустите `TeleVault.exe`.
4. Нажмите `добавить экспорт` и выберите папку экспорта Telegram Desktop.

Можно выбрать одну папку экспорта или общую папку, внутри которой лежит несколько экспортов.

## Что умеет

- читать сообщения выбранной переписки
- показывать фото
- воспроизводить видео
- открывать audio/voice
- открывать файлы
- показывать стикеры
- показывать video notes / кружки
- группировать albums
- искать внутри переписки и переходить к найденному сообщению

## Локально и оффлайн

TeleVault работает с файлами на вашем компьютере. Данные не загружаются в облако, не синхронизируются и не отправляются на внешний сервер.

## Windows

Recommended: Windows 10/11 x64 через основной portable-пакет `TeleVault-v2.9.7.zip`.

Windows 7 SP1 x64: legacy / best effort только через отдельный win7 legacy asset, если он реально опубликован в GitHub release assets для конкретного релиза. Основной пакет для Windows 10/11 не является Windows 7-сборкой.

Не скачивайте отсутствующие DLL, включая `api-ms-win-core-path-l1-1-0.dll`, со случайных сайтов и не копируйте их в `System32` или `SysWOW64`.
